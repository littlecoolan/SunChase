# ☀️ SunChase — 全球追光地图

一个为数字游民设计的实时全球天气地图，帮你找到此刻阳光最充足的城市。

## 功能

- 🗺️ 全球交互式地图，覆盖 60+ 城市
- ☀️ 实时天气数据（via Open-Meteo API）
- 📊 游民指数评分算法（综合温度、天气状况）
- 🇨🇳 中国城市单独分类，支持省份筛选
- 🔍 多维度筛选：全部 / 仅晴天 / 舒适温度 15–25°C / 避开雨天
